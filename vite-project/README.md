Проєкт Electro Shop
Мій проєкт на React це сучасний інтернет магазин електроніки. У ньому я реалізував повний цикл взаємодії користувача з товарами від швидкого пошуку до фінального оформлення замовлення. Основний акцент я зробив на швидкості роботи, інтуїтивній навігації та повній адаптивності під будь-які екрани.

Стек технологій
Для створення фронтенду я використовував React.js та React Router Dom для плавної маршрутизації без перезавантаження сторінок. Візуальну частину побудовано на основі Bootstrap 5 та Bootstrap Icons. Роль бекенду та бази даних виконує Supabase, через який я також налаштував повноцінну систему авторизації. Додатково я інтегрував бібліотеку Swiper.js для створення сучасних слайдерів та React Hot Toast для виведення приємних сповіщень.

Я розробив зручний каталог з поділом техніки на категорії
Користувачі можуть створювати акаунти, авторизуватися, додавати товари в кошик або зберігати їх у список обраного. Для кожного покупця передбачений особистий кабінет, де зберігаються його контактні дані та відображається історія попередніх замовлень. Процес покупки максимально спрощений і завершується на окремій сторінці чекауту з автоматичною валідацією введеної інформації.

Увесь інтерфейс магазину є динамічним. Хедер, сітка товарів та модальні вікна автоматично підлаштовуються під мобільні телефони, планшети та великі монітори комп'ютерів. Структура коду розділена на логічні та перевикористовувані компоненти, що робить проєкт зручним для підтримки та подальшого розвитку.
